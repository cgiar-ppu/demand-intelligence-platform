CGIAR GloMIP Data Extraction Summary
=====================================
Date: 2026-04-14
Source: https://glomip.cgiar.org

OVERVIEW
--------
Successfully extracted real data from the CGIAR Global Market Intelligence Platform (GloMIP)
using session-based CSRF token authentication against their Laravel backend API endpoints.

The platform contains:
- 667 Seed Product Market Segments (SPMS) across 24 crops and 123 countries
- 433 Target Product Profiles (TPPs) across 23 crops and 21 research centers
- 872 unique trait IDs
- 484,438,385 total hectares covered

APPROACH RESULTS
----------------

Approach 1 (Direct portal URLs): PARTIAL SUCCESS
  - Pages load but contain no inline data (all loaded via AJAX)
  - Successfully identified API endpoints from page source

Approach 2 (API endpoints): SUCCESS
  - POST endpoints require CSRF tokens (Laravel framework)
  - Solved by: (1) GET the page to obtain session cookie + CSRF token, (2) POST with both
  - GET endpoints (ms-filters, tpp-filters) work without authentication
  - Working endpoints:
    * GET  /ms-filters           -> crop/country/region filter options
    * GET  /tpp-filters          -> TPP filter options
    * POST /market-segments/overview  -> summary statistics, country data, hectares
    * POST /market-segments/list      -> full list of 667 market segments
    * POST /target-product-profiles/overview -> TPP summary statistics
    * POST /target-product-profiles/list     -> full list of 433 TPPs

Approach 3 (Page source analysis): SUCCESS
  - Found API endpoint URLs in JavaScript AJAX calls
  - Found CSRF token in meta tags
  - No __NEXT_DATA__ (site uses Laravel, not Next.js)

Approach 4 (Direct download patterns): FAILED
  - All pattern URLs return 404

Approach 5 (Public datasets): NO RESULTS
  - CGIAR Dataverse: no GloMIP datasets found
  - GARDIAN: API redirects, no direct datasets
  - No standalone CSV/Excel downloads publicly available

Approach 6 (Bulletins/documentation): PARTIAL
  - 34 Market Intelligence Briefs found (PDF publications via HDL handles)
  - No downloadable raw data in bulletin section
  - Impact Opportunities portal has no discoverable API endpoints (may require login)
  - Investor Dashboard requires authentication for data access

BLOCKED ENDPOINTS
-----------------
  - /impact-opportunities/* -> no API endpoints found (404)
  - /investor-dashboard/* -> requires login
  - /product-catalog/* -> requires login
  - /impacts/* -> requires login
